<a id='nonverbal'></a>
[gildea2018reconstructing](sources.bib?with_internal_ref_link&ref#cldf:gildea2018reconstructing): 366 distinguishes two main formal types of nonverbal predication in Cariban languages: juxtaposition and copular constructions.

* juxtaposed NP + ADV or NP + LOC is found in Arara, Ikpeng, Ye'kwana, Wayana, and Apalaí
* for PC:
    * Nsubj + Npred: nominal (juxtaposition) predication. Limited in functional domains.
    * Nsubj + <span class='smallcaps'>cop</span> + Adverbial (adverbs/postpositional phrases). Fairly unlimited.
* Innovations:
    * Nsubj + <span class='smallcaps'>cop</span> + Npred (S&M 2009)
    * Nsubj + Adverbial

## Observations

### Patterns
* "existential particles" (*mëtë*, *mïntë*, *entë*, *ijtë*)
    * etymologically locatives, used for existential function. they **can** co-occur with the existential negative [Morpheme pirare-nothing](MorphsetTable?no_translation#cldf:pirare-nothing) <a class="exref" example_id="ex-main-neg-part-pirare-cop-nsubj"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>, <a class="exref" example_id="ex-main-neg-part-pirare-nsubj"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>
    * also occur in locative function <a class="exref" example_id="loc-main-aff-part-cop-nsubj"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>, <a class="exref" example_id="loc-main-aff-part-nsubj"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>
    * also combine with postpositional locatives <a class="exref" example_id="histgrme-107"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>, though more often not part of the clause? <a class="exref" example_id="loc-sub-aff-advpred-nsubj-cop"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>, <a class="exref" example_id="convfemgrme-157"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>, <a class="exref" example_id="convfemgrme-99"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>
* the copula...
    * almost always occurs with adverbs
        * counterexample: <a class="exref" example_id="perm-main-q-advpred-nsubj"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>
        * not when negated <a class="exref" example_id="temp-main-neg-nsubj-advpred-jra"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>
    * can occur with locatives and existential particles
        * fairly solid pattern of copula serving as a location for past marking existential function: compare mostly past <a class="exref" example_id="ex-main-aff-part-cop-nsubj"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a> with nonpast <a class="exref" example_id="ex-main-aff-part-nsubj"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>
        * however, no such pattern with locatives: copula <a class="exref" example_id="loc-main-aff-part-cop-nsubj"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a> and no copula <a class="exref" example_id="loc-main-aff-part-nsubj"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a> show no salient distribution
    * almost never occurs with nouns
        * only counterexample: <span class='smallcaps'>np</span><sub><span class='smallcaps'>pred</span></sub> <span class='smallcaps'>cop</span> with 'sick' <a class="exref" example_id="temp-main-q-npred-cop"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>; can occur without <span class='smallcaps'>cop</span>, too <a class="exref" example_id="temp-main-aff-npred-cop"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>
    * does not combine with [Morpheme pirare-nothing](MorphsetTable?#cldf:pirare-nothing)
        * one counterexample with <span class='smallcaps'>part</span><sub><span class='smallcaps'>pred</span></sub> + [Morpheme pirare-nothing](MorphsetTable?no_translation#cldf:pirare-nothing) + <span class='smallcaps'>cop</span> + <span class='smallcaps'>np</span><sub><span class='smallcaps'>subj</span></sub> is with a concessive <a class="exref" example_id="ex-main-neg-part-pirare-cop-nsubj"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a> <span title='@spike: I forgot, what was your analysis for this particular example?'>❓</span>
        * it does occur with [Morpheme pinire-nothing](MorphsetTable?#cldf:pinire-nothing), though <a class="exref" example_id="loc-main-neg-nsubj-cop-pinire-part"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>, <a class="exref" example_id="temp-main-neg-advpred-cop-pinire-nsubj"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a> <span title='@spike: what do you think of the analysis that the copular form is synchronically deverbal in all of these examples, licensing the nominal negator?'>❓</span>
    * is always present in subordinate clauses (*chi-yawë* 'when', *chi-pëke* 'because') <span title='@spike: would you analyze these subordinate clauses as having a copula? if  yes, that would constitute a small but solid difference between main and subordinate clauses, right?'>❓</span>
        * even with nominal predicates: <a class="exref" example_id="cat-sub-aff-npred-nsubj-cop"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>, <a class="exref" example_id="temp-sub-aff-npred-nsubj-cop"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>
        * one example of negation, occurs on additional copula <a class="exref" example_id="loc-sub-neg-locpred-cop-neg-nsubj"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>
        * one example of *chi chipokono* <a class="exref" example_id="convamgu-101"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>
* several negation strategies:
    * [Morpheme jraneg](MorphsetTable?#cldf:jraneg) for adverbs and on nouns in the existential [Morpheme pirare-nothing](MorphsetTable?no_translation#cldf:pirare-nothing) <span class='smallcaps'>np~subj~</span>[Morpheme jraneg](MorphsetTable?no_translation#cldf:jraneg) construction <a class="exref" example_id="ex-main-neg-pirare-nsubj-jra"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>
    * [Morpheme pinire-nothing](MorphsetTable?#cldf:pinire-nothing) for nominal predicates and ones with locative particles
    * [Morpheme pirare-nothing](MorphsetTable?#cldf:pirare-nothing) for existential predicates
        * also for identification? <a class="exref" example_id="id-main-neg-npred-pirare"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>
    * [Morpheme janeg](MorphsetTable?no_translation#cldf:janeg) or [Morpheme jnarineg](MorphsetTable?no_translation#cldf:jnarineg) on the copula 
* order is fairly flexible; potentially rigid <span class='smallcaps'>adv</span>~<span class='smallcaps'>pred</span>~ <span class='smallcaps'>np~subj~</span> <span class='smallcaps'>cop</span>, though negated counterexample <a class="exref" example_id="histyarirdi-249"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>
* unclear role of [Morpheme maniki-rel-anim](MorphsetTable?#cldf:maniki-rel-anim) in <span class='smallcaps'>np</span><sub><span class='smallcaps'>pred</span></sub> + <span class='smallcaps'>np</span><sub><span class='smallcaps'>subj</span></sub> construction
* construction with two copulas *chi wejsapë*? <a class="exref" example_id="convhistfamsjm-92"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>, <a class="exref" example_id="convhistfamsjm-59"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>, <a class="exref" example_id="histgrme-17"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>, <a class="exref" example_id="histgrme-107"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a> <span title='ask natalia'>❗️</span>

* locative predicates are teeming with "existential" particles: largely went by 'estar' vs 'haber' in translation, with the exception of <a class="exref" example_id="convcosnoind-48"cldf="../yawarana-sketch-cldf/cldf/metadata.json" ></a>

## Overview tables
Explanations:

* existential particles are represented as <span class='smallcaps'>part~pred~</span>, even when clearly serving as a locative
* order variation is represented with + between the variable elements
* optional subjects were not counted as resulting in two different constructions
* otherwise identical constructions with and without a copula are counted as different
